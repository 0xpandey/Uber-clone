# 🚗 Uber Landing Page Clone

This project is a **UI clone of the Uber landing page** built using **HTML and CSS**.  
It focuses on recreating the layout, styling, and structure of the Uber homepage while practicing front-end development skills.

## 🌐 Live Preview
You can open the `index.html` file in your browser to view the project locally.

## 📌 Features

- Clean and structured HTML
- Modern CSS styling
- Responsive layout design
- Navigation bar
- Hero section
- Service information sections
- Footer layout

## 🛠️ Built With

- HTML5
- CSS3

## 🎯 Purpose of the Project

This project was created to:

- Practice **HTML and CSS layout building**
- Improve **UI cloning skills**
- Understand **modern website structure**
